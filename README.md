
**Service Booking Web Application**

1. Overview
    A responsive and interactive web application that allows users to browse through various laundry and cleaning services, add them to a dynamic cart, and proceed to book them. The project features a clean, modern UI with a split-screen layout for easy navigation and interaction.

## Features
**Service Browsing:** 
    View services one by one with their respective images and prices.
**Dynamic Cart System:** 
    Add items to the cart or skip to the next service.
**Real-time Price Calculation:** 
    The total amount updates automatically as items are added to the cart.
**State Management:** 
    Empty state handling with intuitive icons and messages when no items are in the cart.
**Conditional Booking Form:** 
    The "Book Now" button remains disabled until at least one item is added to the cart.


# How to Run
1. Extract the `.zip` folder.
2. Ensure `index.html`, `style.css`, and  `image/` folder (containing `logo.png` and `a.jpeg`) are in the exact same directory.
3. Double-click `index.html` to open it in any modern web browser (Chrome, Firefox, Safari, Edge).
